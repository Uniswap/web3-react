# web3-react [![Version](https://img.shields.io/npm/v/@web3-react/core)](https://www.npmjs.com/package/@web3-react/core) [![Downloads](https://img.shields.io/npm/dm/@web3-react/core)](https://www.npmjs.com/package/@web3-react/core) [![Sponsors](https://img.shields.io/github/sponsors/NoahZinsmeister)](https://github.com/sponsors/NoahZinsmeister)

React library for connecting Ethereum wallets.

## Features

- No fuss library for connecting wallets
- Built-in connectors for MetaMask, WalletConnect, and Coinbase Wallet
- Easily write your own wallet connectors
- Connect multiple wallets at once
- TypeScript ready
- Zero-dependencies (besides ethers.js peer dependency)
- MIT License

Learn how to use wallets in your React app [more](/getting-started).
